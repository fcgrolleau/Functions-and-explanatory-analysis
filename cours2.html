Tests statistiques

Continuous Variables

Example

Welch \(t\)-test

\[ t=\frac{\bar{x}_1 - \bar{x}_0}{\sqrt{\frac{s_0^2}{n_0}+\frac{s_1^2}{n_1}}} \]

\[ se(\bar{x}_1 - \bar{x}_0) = \sqrt{{s_0^2}/{n_0}+{s_1^2}/{n_1}} \]

  • 95% Confidence Interval

\[ IC = \bar{x}_1 - \bar{x}_0 \pm t_{\nu, \alpha/2} * se(\bar{x}_1 - \bar{x}_0)\] with \(\nu\) = No. of degrees of freedom.

Pearson's chi-square test

The value of the test-statistic is

\[ \chi^2 = \sum_{i,j} \frac{(O_{ij} - E_{ij})^2}{E_{ij}} \]

where :

  • \(O_{ij}\) = the number of observations in each cell \(i,j\)
  • \(E_{ij}\) = the expected (theoretical) count of type under the null hypothesis

Chi-square test in R

chi <- chisq.test(table(compl$anypro5, compl$group), correct = F)

Confidence interval of the Mean Difference

  • First assume the true variance is the same in both groups
  • Pooled standard deviation : \[s=\displaystyle \sqrt{\frac{(n_0-1)s_0^2 + (n_1-1) s_1^2}{n_0+n_1-2}}\]
  • Standard error \[se(\bar{x}_1 - \bar{x}_0) = s \times \sqrt{\frac{1}{n_0}+\frac{1}{n_1}}\]

Confidence interval of the Mean Difference

Then, if \(X_0\) and \(X_3\) are Gaussian, \(\frac{\bar{x}_1 - \bar{x}_0}{se(\bar{x}_1 - \bar{x}_0)}\) has a \(t\) distribution with \(n_0+n_1-2\) d.f.

The confidence interval is: \[\bar{x}_1 - \bar{x}_0 \pm t_{\alpha/2;n_0+n_1-2} se(\bar{x}_1 - \bar{x}_0)\]

Confidence interval of the Mean Difference in R base

x1bar <- 16.24679
x0bar <- 22.34345
se0 <- 17.01
se1 <- 13.71
n0 <- 140
n1 <- 161

## [1] -6.09666

CI of MD R base

spooled <- sqrt( ((n0-1)*se0**2 + (n1-1)*se1**2) / (n0 + n1 - 2))
sem <- spooled * sqrt(1/n0 + 1/n1)
d + qt(p = c(0.025, 0.5, 0.975), df = n0+n1-2) * sem
## [1] -9.583523 -6.096660 -2.609797

RD, RR and OR

\[RD = p_1 - p_0\]

\[RR = p_1 / p_0\]

\[OR=\displaystyle \frac{p_1/(1-p_1)}{p_0/(1-p_0)}\]

RD, RR and OR

RR <- p1/p0
OR <- (p1 /(1-p1)) / (p0 /(1-p0))
round(c(p0, p1, RD, RR, OR), 3)
## [1] 0.264 0.137 0.128 0.517 0.441

CI of RR and OR

  • \[RR = \frac{a/(a+c)}{b/(b+d)} \]
  • \(\Delta\) \(RR\) is here the sample estimate of the relative risk, not the true population value

CI of RR

  • Log (natural logarithm) transform to compute the CI (Katz, 1978) \[ log(RR) \sim \mathcal{N} (log(\phi), \sigma_{logRR}^{2})\] with \(\phi\) the value of the relative risk in the population et \[\sigma_{logRR}^{2} = \frac{c}{a \times (a + c)} + \frac{d}{b \times (b + d)} \]

  • CI of log(RR) : \[ log(RR) \pm Z_{\alpha/2} \times \sigma_{logRR} \]

  • CI of RR : \[ e^{(log(RR) \pm Z_{\alpha/2} \times \sigma_{logRR})}\]

##   no          139     103

CI of RR with R base

a <- 22
b <- 37
c <- 139
d <- 103

CI of OR

  • Log transform to compute the CI (Woolf, 1955)

\[ log(OR) \sim \mathcal{N} (log(\theta), \sigma_{logOR}^{2})\]

with \(\theta\) the population value of the odds ratio

\[\sigma_{logOR}^{2} = \frac{1}{a} + \frac{1}{b} + \frac{1}{c} + \frac{1}{d} \]

CI of log(OR) : \[ log(OR) \pm Z_{\alpha/2} \times \sigma_{logOR} \]

CI of OR : \[ e^{(log(OR) \pm Z_{\alpha/2} \times \sigma_{logOR})}\]

Relation between OR and RR

  • OR mathematically more interesting (better properties)
  • RR more interpreatble
  • Convert OR in RR, depends on \(p_0\)

\[RR = \displaystyle\frac{OR} {1 - p_0 + (p_0 \times OR)}\] - Grant. BMJ 2014;348:f7450 - If \(p_0\) is small, thedenominator \(\rightarrow\) 1, and \(RR \approx OR\)

Relation between OR and RR (plot)

\[ RR = \frac{OR}{ (1 - p_0 + p_0 \times OR)} \] with \(p_0\) the baseline risk
